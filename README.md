# RNA-seq Analysis of Triticum aestivum under Salt Stress
Analysis of transcriptomic responses of wheat to salinity stress using publicly available RNA-seq data.

## Project Overview
This project investigates transcriptional changes in wheat
(Triticum aestivum) exposed to salt stress.

Dataset:
- GEO accession: GSE225565

Tools:
- R
- dplyr
- tidyr
- ggplot2
